## Image Guidelines

- All images should be optimized for web (compressed)
- Hero images should be at least 1920px wide
- Logo images should have transparent backgrounds
- Book cover images should be high-quality and consistent in size (recommended: 800x1200px)
- Maintain aspect ratios appropriate for the content
- Consider accessibility by ensuring proper contrast with overlaid text

You can replace these placeholders with actual high-quality images related to ISKCON. Good sources for ISKCON-related images include:

1. Official ISKCON websites
2. ISKCON photo archives
3. Free stock photo sites (for generic spiritual imagery)
4. Commissioned photography of ISKCON temples and activities
5. The Bhaktivedanta Book Trust (BBT) for official book covers: https://www.krishna.com/

Ensure you have proper permissions or rights to use any images included in the website.
